About - Features

  • This project will give consumers a fully functional user interface and let them design a space that is entirely suited to their tastes.
  • With a fully adjustable and user-friendly user interface, they can also use this to adapt the surroundings into their hand.
  • Each device's state is tracked by this system, which also keeps us informed about the overall condition of our house.
  • Enables the users to have private cloud storage space while serving as a host for the controller. Thanks to the host server that will be built in each home.
  • This results in the increase of overall efficiency of the smart home system by using the additionally provided data gathered for error detection.

    The Dashboard

    A dashboard is very important in a car. It is just as important in any smart home system to display the status of the devices that are a part of the smart home system.


    Purpose:


    Avisual display of all of your data. While it can be used in all kinds of different ways, its primary intention is to provide information at-a-glance, such as KPIs. A dashboard usually sits on its own page and receives information from a linked database.


    Function:


    A dashboard is an information management tool that tracks the condition of your business, a department, a campaign, or a specific process by tracking key performance indicators (KPIs), metrics, and vital data.
